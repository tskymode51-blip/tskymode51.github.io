Ubuntu 24.04 LTSをUSBから簡単インストール

ステップ1: Ubuntu ISOをダウンロード

まずはUbuntuの公式サイトからISOファイルをダウンロードします。

  • Ubuntu公式ダウンロードページにアクセス
  • 「Download Ubuntu 24.04 LTS」を選択し、ISOファイルを保存

ステップ2: ブータブルUSBを作成

USBをUbuntuインストール用に準備します。

  • Windowsの場合: Rufus
  • Windows/Mac/Linuxの場合: Etcher

手順:

  1. USBを接続
  2. Rufus/Etcherを起動
  3. ISOを選択
  4. USBを選択し「書き込み」または「Flash」をクリック
  5. 完了まで待つ

ステップ3: PCをUSBから起動

USBを挿したままPCを再起動し、ブートメニューでUSBを選択します。

  • 多くのPCでは F12, F2, Esc, Del などで起動メニューを表示
  • USBを選択して起動
BIOS/UEFIで起動優先順位を変更する必要がある場合もあります。

ステップ4: Ubuntuをインストール

インストーラーに従ってUbuntuをインストールします。

  1. 言語選択
  2. 「Ubuntuをインストール」をクリック
  3. キーボードレイアウト選択
  4. インストールタイプを選択
    • 標準インストール（推奨）
    • 最小インストール（軽量）
  5. ディスクの割り当てを設定
    • 「ディスクを削除してUbuntuをインストール」
    • 「その他」から手動パーティション設定も可能
  6. ユーザー名・パスワード設定
  7. 「インストール」をクリック

ステップ5: 初期設定とアップデート

Ubuntu起動後、まずは最新状態にします。

sudo apt update
sudo apt upgrade -y
インストール中にディスクを削除するとデータが消えるため、事前にバックアップをおすすめします。
