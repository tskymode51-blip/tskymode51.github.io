Ubuntu × OBD2 でクルマ診断！

ダイハツネイキッド & トヨタiQ でテスト

🏠 ホーム

こんにちは！今回はUbuntuノートPCを使って、
クルマ（ダイハツ ネイキッド / トヨタ iQ）の
故障診断やリアルタイムデータ表示に挑戦しました 🚗💨

🔧 準備したもの

  • Ubuntu ノートPC
  • USB OBD2 アダプター
  • Python + OBD ライブラリ

💻 セットアップ手順

  1. Ubuntu に Python 環境を準備
  2. 「python-OBD」ライブラリをインストール
  3. USB-OBD2 アダプターを車に接続
  4. ターミナルで接続確認！

📊 表示できるデータ

エンジン回転数、速度、水温、バッテリー電圧などが
リアルタイムで見られます！

🚙 感想

最初はちょっと難しかったけど、
データが数字で動くのを見るとテンション上がります！
故障診断コードも見れるのでDIY整備にも便利✨
